--Описание программы--
К ToolBar добавлены кнопки (в порядке их появления с левого края):
1. New - очистка всех зон;
2. Open - загрузить картинку;
3. Save - сохранить поле (на данный момент не реализовано);
4. Select - выбрать область (на данный момент отключено);
5. Negative - инвертирующий фильтр;
6. Monochrome - фильтр, делающий картинку ч/б;
7. Watercolor - фильтр акварелизации;
8. Gamma - гамма-фильтр;
9. Sharper - фильтр, увеличивающий резкость;
10. Stamping - фильтр - тиснение.
11. Magnifier - увеличение;
12. Blur - фильтр-блюр;
13. Rotate - фильтр, поворачивающий картинку;
14. About - о программе.

Что не работает на данный момент:
1. Объёмная визуализация - нет, поскольку основной функцилнал не доработан;
2. Select:
    a. Сам select работает как положено и нормально копирует область из A в B, но квадрат-рамка, показывающий, какую область зоны A, отображается некорректно во всех случаях;
    b. Ввиду a на данный момент select активирован постоянно;
3. Есть вопросы по дизерингу (оба два);
4. Не реализованы дифференцирующие фильтры;
5. Нет сохранения;
6. Прочие скрытые баги (результат необычного поведения и т.д.).
7. Toolbar не повторяется на Menubar.
